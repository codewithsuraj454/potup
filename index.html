What is Potup?

Potup is a modern social media platform built for open, public conversations. It allows users to create posts, reply to discussions, and share content that can be seen by anyone around the world. The goal of Potup is to make communication simple, fast, and globally accessible.

Key Features

  • Create public posts visible to everyone
  • Advanced reply and repost system
  • Personalized global content feed
  • Fast and intuitive interface
  • Accessible from anywhere in the world
  • Real-time notifications
  • Rich media support
